# 🏦 Bank Customer Churn Prediction

## 📌 Project Overview

A Machine Learning project that predicts whether a bank customer is likely to churn using a **Random Forest Classifier**. The application provides an interactive dashboard built with **Streamlit**, allowing users to input customer details and receive real-time churn predictions.

---

## 🌐 Live Demo

👉 **[View Live Application](https://bank-customer-churn-analysis-8tvhnprqppstb4uqyfcw57.streamlit.app/)**

---

## 📊 Dataset

- 10,000 customer records
- Features include:
  - Credit Score
  - Geography
  - Gender
  - Age
  - Tenure
  - Balance
  - Number of Products
  - Has Credit Card
  - Is Active Member
  - Estimated Salary
- Target Variable: **Exited (Churn / Not Churn)**
