# 🏦 Bank Customer Churn Prediction

## 📌 Project Overview
A Machine Learning project that predicts whether a bank customer is likely to churn using a Random Forest Classifier.

## 📊 Dataset
- 10,000 customer records
- Features include Credit Score, Age, Balance, Geography, Products, etc.

## 🚀 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn
